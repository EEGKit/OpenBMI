Beim d2-Test geht es darum, alle d's mit zwei Strichen zu
identifizieren. Dabei spielt es keine Rolle, wo sich die beiden
Striche befinden. Erscheint auf dem Bildschirm ein "d" mit zwei
Strichen, drücken Sie bitte so schnell wie möglich auf das "J" ihrer
Tastatur. "J" steht dabei für "Ja bzw. richtig". Erscheint auf dem
Bildschirm ein "d" mit einer anderen Anzahl von Strichen (0, 1, 3 oder
4) oder ein "b" (mit beliebiger Anzahl von Strichen), müssen Sie "F
auf Ihrer Tastatur drücken. Das "F" steht demnach für "falsch".

Es ist wichtig, dass Sie auf jeden Reiz eine Antwort geben und dass
Sie die Tasten so schnell wie möglich drücken.

Zunächst kommt ein Probedurchgang!
