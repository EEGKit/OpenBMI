\bf{Visuelles Oddball Experiment}\rm

Sitzen Sie bitte möglichst entspannt, die Unterarme auf den Tisch
gestützt und die Zeigefinger auf den Tasten \bf{F}\rm und \bf{J}\rm.

Im Experiment werden Sie eine Folge von Bilder sehen. Auf jedem Bild
ist ein einzelnes Objekt dargestellt, dass entweder eine Waffe oder
ein harmloser Gegenstand ist.

Haben Sie eine Waffe erkannt, drücken Sie bitte möglichst schnell die
Taste \bf{J}\rm, andernfalls die Taste \bf{F}\rm.
