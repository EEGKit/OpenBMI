Detailed Description

This struct provides information about the eyetracking system in use.

The struct contains the following information: samplerate: sample rate of eyetracking system in use iV_MajorVersion: major version number of iView X in use iV_MinorVersion: minor version number of iView X in use iV_Buildnumber: build number of iView X in use API_MajorVersion: major version number of iView X SDK in use API_MinorVersion: minor version number of iView X SDK in use API_Buildnumber: build number of iView X SDK in use iV_ETDevice: type of eyetracking device

To update information in "SystemInfoStruct" use function "iV_GetSystemInfo".
