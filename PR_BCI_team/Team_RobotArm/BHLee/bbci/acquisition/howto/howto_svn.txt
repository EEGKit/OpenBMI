To get permission to access the BBCI - SVN, you must register here
(please choose FirstnameLastname as user name, e.g. BenjaminBlankertz;
password is up to you, of course):
  https://ml01.zrz.tu-berlin.de/cgi-bin/twiki/view/TWiki/TWikiRegistration


** Windows:
Install 'Tortoise-SVN':
  http://tortoisesvn.net/downloads
Create a folder 'svn' and subfolders 'bbci' and 'ml' therein. Right-click
in windows explorer and select 'SVN checkout':
URL of repository:
  https://ml01.zrz.tu-berlin.de/svn/ida/public/bbci
As checkout directory choose the folder created as stated above
(e.g. d:\svn\bbci).
-> Press OK
Authentification Window pops up:
Enter your TWiki username and password.
And do the same for a second repository:
  https://ml01.zrz.tu-berlin.de/svn/ida/public/ml
to folder 'd:\svn\ml'.

** Linux:
Install Subversion package, e.g. in Debian
sudo aptitude install subversion
Then do the following (replace UserName with your TWiki username):
mkdir ~/svn
cd ~/svn
svn checkout --username UserName https://ml01.zrz.tu-berlin.de/svn/ida/public/bbci ida/public/bbci
svn checkout --username UserName https://ml01.zrz.tu-berlin.de/svn/ida/public/ml ida/public/ml


** All platforms:
To load the new version of the toolbox in Matlab do:
cd('~/svn/ida/public/bbci/toolbox/startup');
startup_bbci;

You may want to include those lines in your local startup.m.
