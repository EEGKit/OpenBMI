Install: 
 PuttyGen http://www.putty.nl/download.html
 WinCVS http://www.wincvs.org/download.html
 Python http://www.python.org/download/

PuttyGen -> generate private/public key
save both somewhere, in this example the private key is
  d:\_ssh\key_rsa_1024_private.ppk

convert public key
 (cut head and tail such that it starts with 'AA' and ends with '=',
  prepend 'ssh-rsa ' and append USER@MACHINE, e.g., 'charite@neuropc')
and winscp it to the linux file system
 append it to .ssh/authorized_keys of neuro_cvs@trumpet
 append it to .ssh/authorized_keys of neuro_toolbox@trumpet


WinCVS:

Menu: Remote | Checkout Module

(1)
Module name: matlab/bci
Local folder to checkout to: C:\neuro_cvs
CVSROOT:
:ssh;privatekey='d:\_ssh\key_rsa_1024_private.ppk';user=neuro_cvs;hostname=cvs-ida.first.fhg.de:/home/cvs/neuro_cvs/repos

(2)
Module name: matlab
Local folder to checkout to: C:\neuro_toolbox
CVSROOT:
:ssh;privatekey='d:\_ssh\key_rsa_1024_private.ppk';user=neuro_toolbox;hostname=cvs-ida.first.fhg.de:/home/cvs/neuro_toolbox/repos
